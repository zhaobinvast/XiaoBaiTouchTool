# XiaoBaiTouchTool 使用说明

一款 macOS 触控板手势映射工具，让你用多指手势快速启动应用、控制系统，告别繁琐的鼠标点击。

## 系统要求

- macOS 13.0 (Ventura) 及以上
- 带触控板的 Mac（MacBook 内置触控板或 Magic Trackpad）

---

## 安装

1. 下载 `XiaoBaiTouchTool.zip`，解压得到 `XiaoBaiTouchTool.app`
2. 将 `XiaoBaiTouchTool.app` 拖入「应用程序」文件夹
3. 双击打开

> 如果系统提示「无法打开，因为它来自身份不明的开发者」，请前往「系统设置 → 隐私与安全性」，点击「仍要打开」。

---

## 首次启动

打开应用后，系统会弹出辅助功能权限请求：

1. 点击「打开系统设置」
2. 在「隐私与安全性 → 辅助功能」中找到 XiaoBaiTouchTool，打开开关
3. 返回应用，手势监听自动启动

授权完成后，应用以**菜单栏图标**形式运行，不会出现在 Dock 中。首次启动会自动开启「开机自启动」，下次登录无需手动打开。

---

## 菜单栏

点击菜单栏中的 XiaoBaiTouchTool 图标，可以看到：

| 菜单项 | 说明 |
|--------|------|
| 已启用 ✓ / 已禁用 | 当前手势监听状态 |
| 启用 / 禁用手势监听 | 临时开关手势功能 |
| 开机自启动 | 勾选后登录时自动启动 |
| 设置... (⌘,) | 打开手势配置窗口 |
| 退出 (⌘Q) | 退出应用 |

---

## 配置手势映射

### 添加映射

1. 点击菜单栏图标 → **设置...**
2. 点击右上角 **+** 按钮
3. 在向导中选择**手势**（如「三指轻点」）
4. 选择**动作**：
   - 应用操作：选择目标应用 + 操作方式（打开/隐藏/切换）
   - 系统指令：直接选择系统功能（锁屏、截图、调节音量等）
5. 点击保存，立即生效

### 编辑映射

点击设置列表中的任意条目即可修改。

### 删除映射

在条目上点击删除按钮即可移除。

## 常见问题

**手势没有反应？**
检查菜单栏状态是否显示「已启用 ✓」，并确认辅助功能权限已授予。

**如何临时关闭手势？**
点击菜单栏图标 → 禁用手势监听，需要时再重新启用。

**如何取消开机自启动？**
点击菜单栏图标 → 取消勾选「开机自启动」。

**配置文件在哪里？**
手势映射保存在 `~/Library/Application Support/XiaoBaiTouchTool/mappings.json`，可手动备份或迁移。
