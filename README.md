# XiaoBaiTouchTool

一款轻量级 macOS 触控板手势映射工具，可将多指手势绑定到应用操作或系统快捷指令。无需复杂配置，开箱即用。

## 系统要求

- macOS 13.0 (Ventura) 及以上
- Apple Silicon 或 Intel Mac（需带触控板）

## 安装

### 从源码构建

```bash
git clone <repo-url>
cd XiaoBaiTouchTool
./build_app.sh
open XiaoBaiTouchTool.app
```

构建脚本会自动完成编译、创建 .app 包并复制图标资源。

### 首次启动

1. 系统会弹出辅助功能权限请求，前往「系统设置 → 隐私与安全性 → 辅助功能」中允许 XiaoBaiTouchTool
2. 授权后手势监听自动启动，无需重启应用
3. 应用以菜单栏图标形式运行，不会出现在 Dock 中
4. 首次启动默认开启「开机自启动」

## 使用方法

### 菜单栏

点击菜单栏图标可以：

- 查看当前状态（已启用 ✓ / 已禁用）
- 启用 / 禁用手势监听
- 开关开机自启动
- 打开设置窗口（⌘,）
- 退出应用（⌘Q）

### 添加手势映射

1. 点击菜单栏图标 → 设置
2. 点击右上角 `+` 按钮
3. 选择手势类型
4. 选择要执行的动作（应用操作或系统指令）
5. 保存即生效，无需重启

### 编辑 / 删除映射

- 点击已有映射条目可编辑
- 在条目上可删除

## 支持的手势

### 双指（7 种）

| 手势 | 说明 |
|------|------|
| 双指轻点 | 两指同时轻触触控板 |
| 双指点按 | 两指同时按压触控板 |
| 双指上滑 | 两指向上滑动 |
| 双指下滑 | 两指向下滑动 |
| 双指左滑 | 两指向左滑动 |
| 双指右滑 | 两指向右滑动 |
| 双指轻点两下 | 两指快速轻触两次 |

### 三指（6 种）

| 手势 | 说明 |
|------|------|
| 三指轻点 | 三指同时轻触 |
| 三指点按 | 三指同时按压 |
| 三指轻点两下 | 三指快速轻触两次 |
| 三指左指轻点 | 三指触控板时，左侧手指单独轻点 |
| 三指中指轻点 | 三指触控板时，中间手指单独轻点 |
| 三指右指轻点 | 三指触控板时，右侧手指单独轻点 |

### 四指（6 种）

| 手势 | 说明 |
|------|------|
| 四指轻点 | 四指同时轻触 |
| 四指点按 | 四指同时按压 |
| 四指轻点两下 | 四指快速轻触两次 |
| 四指左指轻点 | 四指触控板时，左侧手指单独轻点 |
| 四指中指轻点 | 四指触控板时，中间手指单独轻点 |
| 四指右指轻点 | 四指触控板时，右侧手指单独轻点 |

## 支持的动作

| 动作 | 说明 |
|------|------|
| 打开 / 激活应用 | 启动或切换到指定应用 |
| 隐藏应用 | 隐藏指定应用窗口 |
| 切换显示/隐藏 | 在显示和隐藏之间切换 |

### 系统指令

| 动作 | 说明 |
|------|------|
| 锁屏 | 锁定屏幕 |
| 睡眠 | 系统进入睡眠 |
| 关机 | 关闭电脑 |
| 重启 | 重启电脑 |
| 推出所有磁盘 | 推出所有可推出的磁盘 |
| 显示桌面 | 显示桌面（F11） |
| 启动台 | 打开 Launchpad（F4） |
| 调度中心 | 打开 Mission Control（F3） |
| 截取全屏 | 截取整个屏幕 |
| 截取区域 | 截取选定区域 |
| 截取窗口 | 截取指定窗口 |
| 增加/降低亮度 | 调节屏幕亮度 |
| 增加/降低音量 | 调节系统音量 |
| 静音 | 切换静音 |
| 播放/暂停 | 媒体播放控制 |
| 上一首/下一首 | 切换媒体曲目 |

## 数据存储

手势映射配置保存在：

```
~/Library/Application Support/XiaoBaiTouchTool/mappings.json
```

## 技术栈

- Swift 5.9 + SwiftUI
- MultitouchSupport.framework（私有框架，用于触控板原始事件）
- Swift Package Manager 构建
